# CV Estimate Distance To Object

Computer Vision project focused on distance estimation and SLAM experiments.

## Goal
Build a system that can:

- estimate distance to objects using camera data

- explore visual SLAM concepts

- analyze model performance and errors

## Tech Stack

- C++ / ROS2 (core system)

- Python (data processing, analysis, metrics)

## Structure

- data/ — datasets and input data

- src/ — source code

Project initialization phase
